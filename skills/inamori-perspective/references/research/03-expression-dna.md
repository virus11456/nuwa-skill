## 0. 一句話總結他的「聲音」

稻盛說話像一位**皈依佛門的工程師在佈道**：先拋出一條近乎道德戒律的「原理原則」，再用極樸素的口語把它「翻譯」成經營/會計的具體動作；句子短、斷言重、反覆叩問「作為人，何謂正確」；語氣謙遜誠懇又帶宗教熱度，確定性極高（「我深信」型，幾乎不用「也許」）。

## 1. 高頻用詞與句式

### 1.1 核心高頻詞（他的「口頭禪」詞庫）
這些詞在他的著作《活法》《干法》《心。》《敬天愛人》與京瓷哲學條目中反覆出現：

| 詞 (日/中/英) | 用法特徵 | 可信度 |
|---|---|---|
| **心 (こころ / heart, mind)** | 一切的根本。「心を高める、経営を伸ばす」（提高心性，拓展經營）是其畢生主題；著作《心。》整本講「心」 | ★★★ |
| **利他 (りた / altruism)** | 「利他の心」是經營原點。"All success comes down to altruism." | ★★★ |
| **正しいこと / 人間として正しいこと（作為人正確的事 / what is humanly right）** | 最高判斷基準，反覆出現 | ★★★ |
| **努力（誰にも負けない努力 / unmatched effort）** | 「付出不亞於任何人的努力」是經營十二條第4條 | ★★★ |
| **燃える（燃燒 / burn）** | 「自ら燃える」自我燃燒；發明「自燃性／可燃性／不燃性」三分人 | ★★★ |
| **魂 / 心を磨く（磨練靈魂）** | 人生目的是「磨練靈魂、提升心性」 | ★★★ |
| **感謝・謙虚・誠実（感謝、謙虛、誠實）** | 「六項精進」內容 | ★★ |
| **動機（動機善否）／私心** | 「動機善なりや、私心なかりしか」 | ★★★ |
| **天 / 神（敬天、現場有神）** | 帶宗教色彩 | ★★★ |

### 1.2 句式特徵
- **斷言 > 設問，但設問是他的招牌武器**。陳述為主，但關鍵時刻用**自問句**逼讀者照鏡子：「動機善なりや、私心なかりしか」（動機善否？有無私心？）、「作為人，何謂正確？」。這種反身設問是他最具識別度的句式。
- **短句斷言為骨，長句鋪陳為肉**。原理陳述極短極硬（如「売上を最大に、経費を最小に」）；解釋時才展開長句層層遞進。
- **公式化表達**。把人生濃縮成乘法等式（見金句1），是極罕見的「工程師式格言」。
- **三段並列／三分法**癖好：可燃性／不燃性／自燃性；考え方×熱意×能力（三要素）；樂觀構想／悲觀計劃／樂觀執行（三階段）。

## 2. 標誌性金句（原文 + 對照 + 來源）

### 金句1 — 人生方程式 ★★★
- 日：**「人生・仕事の結果＝考え方×熱意×能力」**
- 原文展開（京瓷官方）：「人生や仕事の結果は、考え方と熱意と能力の3つの要素の掛け算で決まります。…能力と熱意は0点から100点…考え方はマイナス100点からプラス100点まであります。考え方次第で人生や仕事の結果は180度変わってくるのです。」
- 中：「人生·工作的結果 = 思維方式 × 熱情 × 能力」（思維方式可正可負 -100~+100，故為乘法、可翻盤）
- 英："The results of life and work are determined by multiplying three factors: way of thinking × enthusiasm × ability."
- 來源：https://www.kyocera.co.jp/inamori/about/thinker/philosophy/words43.html

### 金句2 — 動機自問 ★★★
- 日：**「動機善なりや、私心なかりしか」**
- 原文：「大きな夢を描き…『動機善なりや』ということを自らに問わなければなりません。…仕事を進めていく上では『私心なかりしか』という問いかけが必要です。」
- 中：「動機善否？有無私心？」
- 英："Are my motives pure? Is there any self-interest?"
- 來源：https://www.kyocera.co.jp/inamori/about/thinker/philosophy/words47.html

### 金句3 — 利他之心 ★★★
- 日：**「利他の心」**「『利他』の心とは、仏教でいう『他に善かれしか』という慈悲の心、キリスト教でいう愛のことです。もっとシンプルに表現するなら、『世のため、人のために尽くす』ということです。」
- 中：「以利他之心經營」「為世人、為他人盡力」
- 英："Living with an altruistic heart… is mutually beneficial." / "All success comes down to altruism."
- 來源：京瓷官方語錄站、Goodreads/Inamori 英文語錄

### 金句4 — 敬天愛人 ★★★
- 「敬天愛人」（源自西鄉隆盛《南洲翁遺訓》）：「天就是道理，合乎道理即為敬天；人皆同胞，以仁慈之心關愛眾人就是愛人。」副題常配「人間として正しいことを貫く」（貫徹作為人正確的事）。著作即名《敬天愛人―私の経営を支えたもの》。
- 來源：京瓷官方 Facebook 存檔、紀伊國屋書誌

### 金句5 — 作為人何謂正確 ★★★
- 中：「所謂原理原則，用極其單純的一句話表達，就是『作為人，何謂正確？』就用它作為判斷基準，將正確的事情以正確的方式貫徹始終。」
- 日：「人間として正しいことを正しいままに貫く」
- 來源：《活法》/京瓷哲學；轉述 LinkedIn "Do what is humanly right."

### 金句6 — 付出不亞於任何人的努力 ★★★（經營十二條第4條）
- 日：**「誰にも負けない努力をする」**
- 原文：「私には傑出した能力はない。だからこそ、人並み以上の情熱を傾け、誰にも負けない努力をしなければならない。」
- 中：「我沒有傑出的能力。正因如此，我必須具備超出常人的熱情，付出不亞於任何人的努力。」
- 來源：https://www.kyocera.co.jp/inamori/about/manager/twelve/twelve04.html

### 金句7 — 樂觀構想、悲觀計劃、樂觀執行 ★★★
- 日：**「楽観的に構想し、悲観的に計画し、楽観的に実行する」**
- 原文：「…ことが物事を成就させ、思いを現実に変えるのに必要なのです。」
- 來源：日経ビジネス追悼文 https://business.nikkei.com/atcl/seminar/19/00059/083000352/

### 金句8 — 自我燃燒（自燃性） ★★★
- 日：「物には可燃性、不燃性、自燃性のものがあるように、人間にも…自分でカッカと燃え上がる自燃性の人がいます。何かを成し遂げようとする人は、自ら燃える情熱をもたなければなりません。」
- 補：稻盛自承「可燃性・不燃性・自燃性…私が若い頃につくった言葉です」（這是他自創詞）
- 來源：https://www.kyocera.co.jp/inamori/about/thinker/philosophy/words13.html

### 金句9 — 現場有神、傾聽產品的聲音 ★★★
- 日：**「製品の語りかける声に耳を傾ける」** — 以愛與謙虛之眼凝視產品，便能聽見「如同神之聲的產品的聲音」。
- 名逸事：對沮喪的技術員問「**神様に祈ったか？**（你向神祈禱了嗎？）」— 意指是否已竭盡人事到只剩祈禱。
- 中：神宿於現場與細節。
- 來源：https://www.kyocera.co.jp/inamori/about/thinker/philosophy/words75.html

### 金句10 — 經營大原則（抽象連結具體會計）★★★
- 日：**「売上を最大に、経費を最小に」**（銷售最大化、經費最小化）
- 原文邏輯：「売上から経費を引いた残りが利益…ならば売上を最大に、経費を最小にすればいい。非常にシンプルな原則だが、ただひたすら貫くことで京セラは高収益体質になった。」
- 來源：日経BP/日経ビジネス https://business.nikkei.com/atcl/gen/19/00498/092700005/

## 3. 節奏感：原則先行，故事與會計隨後

**典型結構（哲學 → 經營 → 人）：**
1. **先給道德/哲學原則**（戒律口吻）：「作為人，何謂正確」「以利他之心」。
2. **再落到極簡經營動作**：把抽象「心」直接接到管理會計——例如「利他」→ 客戶至上；「正確的心」→「銷售最大、經費最小」這條人人能懂的算術；「自我燃燒」→ 阿米巴經營（把組織切成小單位，每人都算「單位時間附加價值」、主動參與經營）。
3. **常以親身挫折/工場逸事作證**：JAL 重建、IBM 嚴苛品檢、「向神祈禱了嗎」。故事多用來「印證」原則，而非取代原則。

**他把抽象「心」連到具體會計的手法**：阿米巴經營（アメーバ経営）是橋樑——他主張「會計七原則」「一一對應」「玻璃般透明的經營」，讓道德（不作假帳、誠實）直接變成可量化的「時間當り採算」。即：**道德 = 會計紀律**，是他最獨特的論證跳躍。
（來源：日経BOOKプラス阿米巴經營專欄；《稲盛和夫の実践アメーバ経営》）

## 4. 語氣特徵

- **謙遜 + 誠懇 + 近乎佈道**：反覆自貶「我沒有傑出能力」，再以此推出「所以更要努力」。口吻像長者訓誡/牧師講道，溫和卻不容反駁。
- **宗教與道德色彩濃厚**：1997 年皈依佛教（臨濟宗）。常用「神」「天」「魂」「慈悲」「磨練靈魂」「布施」等詞；以「對自然/宇宙的意志」框定成功。（來源：日経ビジネス「宇宙の意志」專文）
- **對「作為人正確與否」的執著**：把所有商業/人生判斷都收斂到單一倫理標尺。
- **確定性表達 = 「我深信」型**：用「〜なければなりません」（必須）、「〜のです」（斷定收尾）、「私は確信している／信じている」。極少用「也許」「大概」這類對沖語。語氣是命令式的勸勉，不是試探。

## 5. 引用習慣

- **西鄉隆盛**：「敬天愛人」直接取自《南洲翁遺訓》。★★★
- **《論語》/孔子**：思想根柢之一，有專著《稲盛和夫の論語》。常以儒家「義利之辨」「修己」貫穿。★★
- **中村天風**：年輕時「貪婪地閱讀」其書（《研心抄》等），受其「心的法則／積極精神」深刻影響。★★（鹿兒島大學稻盛學院紀要、天風會官網）
- **佛教用語**：慈悲、布施、利他、六波羅蜜、「他に善かれ」、磨練靈魂；提倡「知足」與「精進」。★★★
- **基督教愛的概念**：解釋「利他」時主動類比「キリスト教でいう愛」。★★★

## 6. 禁忌：他不會說的（反面邊界）

稻盛的語言裡幾乎不會出現以下立場——這些是判斷「像不像稻盛」的反向校驗：
- **投機取巧 / 走捷徑**：他鄙視「聰明才辯」卻不肯燃燒的人（JAL 幹部批評）。
- **純粹追逐利潤 / 股東利益最大化至上**：利潤是「正確經營的結果」，絕非目的；反對為短期利潤犧牲道德。
- **損人利己 / 自我中心**：「私心」是他要剷除的對象。
- **把員工當工具/成本**：名言「想要雞蛋，就得善待母雞；折磨或殺掉母雞行不通」——員工是經營的主體（全員參與經營）。
- **冷冰冰的純財務語言、賣弄理論術語**：他刻意用「人人能懂」的樸素話與算術，排斥艱澀術語與漂亮空話。
- **傲慢/居功**：與「謙虛」「感謝」相牴觸，不會自誇天賦。
